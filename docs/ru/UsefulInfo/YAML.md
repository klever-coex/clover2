# YAML — краткая памятка

YAML (YAML Ain't Markup Language) — человекочитаемый формат данных. В Clover2 используется для описания карт ArUco-маркеров, конфигов камер и параметров.

## Основные правила

- **Ключ-значение:** `ключ: значение` (пробел после двоеточия обязателен)
- **Комментарии:** `#` — всё до конца строки игнорируется

  ```yaml
  name: test          # это комментарий
  # а эта строка целиком — комментарий
  ```

  Комментарии можно ставить только в конце строки или на отдельной строке. Блочных комментариев (/* */) в YAML нет.
- **Отступы:** только пробелы (2 или 4), табуляция запрещена
- **Строки:** без кавычек, если нет спецсимволов. Иначе в двойных `" "` или одинарных `' '`
- **Числа:** `42`, `3.14`, `-1.5`, `1e-3`
- **Булевы:** `true` / `false`, `yes` / `no`, `on` / `off`
- **Null:** `~`, `null`, или пустое поле
- **Список (блочный):** строки с `- ` (дефис + пробел):

  ```yaml
  markers:
    - id: 0
    - id: 1
  ```

- **Список:** квадратные скобки, через запятую:

  ```yaml
  markers: [{id: 0, size: 0.3}, {id: 1, size: 0.3}]
  # или
  markers: [0, 1, 2, 3]
  ```

- **Список словарей (блочный, явный):**

  ```yaml
  markers:
    - {id: 0, size: 0.3, x: 1.0, y: 2.0}
    - {id: 1, size: 0.3, x: 3.0, y: 4.0}
  ```

- **Многострочные массивы без ключей:**

  ```yaml
  ids:
    - 10
    - 20
    - 30
  ```

- **Словарь (вложенный):** отступы показывают вложенность:

  ```yaml
  pose:
    x: 1.0
    y: 2.0
  ```

- **Словарь:** фигурные скобки, через запятую:

  ```yaml
  pose: {x: 1.0, y: 2.0, z: 0.5}
  ```

:::{note}
Отступы должны быть согласованными. Если первый уровень с 2 пробелами, все уровни — с 2. Ошибка отступа = программа не сможет прочитать файл.
:::
