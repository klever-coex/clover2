«Клевер» — это современная учебно-исследовательская платформа программируемого квадрокоптера, предназначенная для быстрого старта, обучения и разработки автономных дронов.

Платформа сочетает надёжное современное «железо» и актуальный программный стек. В её основе — производительный полётный контроллер на базе STM32H7 с PX4, обеспечивающий высокую частоту вычислений и стабильную работу алгоритмов управления, а также Raspberry Pi 5 в качестве бортового компьютера для запуска ROS 2-узлов, компьютерного зрения и пользовательской логики.

Обновлённый Клевер делает основной упор на современную робототехническую экосистему:
- ROS 2 — для построения масштабируемых, распределённых и надёжных систем управления
- Docker — для изоляции окружений, воспроизводимых сборок и удобного развёртывания
- Готовые контейнеры и шаблоны проектов — чтобы начать работу за минуты
- Открытый исходный код для обучения и кастомизации

Такой подход делает платформу одинаково удобной как для новичков, которые благодаря преднастроенному образу системы, примерам и подробной документации могут быстро запустить первый полёт и начать программировать без сложной настройки, так и для продвинутых разработчиков, которым требуется стабильная и надёжная база для реализации сложных проектов — автономной навигации, задач компьютерного зрения, разработки роёв дронов, а также исследовательских и промышленных решений.

В комплект входит полностью настроенный образ для Raspberry Pi со всем необходимым ПО, библиотеками и инструментами разработки. Всё программное обеспечение и документация открыты и доступны на [GitFlic](https://gitflic.ru/project/klever-coex/clover2).
