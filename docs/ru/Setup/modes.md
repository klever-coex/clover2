# Полетные режимы

**Режим** полетного контроллера PX4 определяет, как квадрокоптер реагирует на команды с пульта. Режим обычно переключается одним из тумблеров на аппаратуре управления.

Чтобы настроить полетные режимы:

1. В QGroundControl перейдите в панель *Vehicle Setup*.
2. Выберите меню *Flight Modes*.
3. Установите *Mode Channel* на переключатель **SwC** (*Channel 6*).
4. При необходимости установите *Emergency Kill Switch Channel* на переключатель **SwA** (*Channel 5*).
5. Выберите нужные полетные режимы.

Рекомендуемые режимы:

* *Flight Mode 1*: **Stabilized**.
* *Flight Mode 4*: **Altitude**.
* *Flight Mode 6*: **Position**.

После настройки проверьте переключение режимов на пульте.

## Ручное управление

В ручных режимах пилот управляет квадрокоптером напрямую. GPS, компьютерное зрение и барометр не используются. Для таких режимов нужны хорошие навыки пилотирования.

* **STABILIZED** / **MANUAL** — стабилизация горизонтального положения. Пилот управляет газом, наклоном по тангажу и крену, а также угловой скоростью по рысканью.
* **ACRO** — управление газом и угловыми скоростями по тангажу, крену и рысканью. Используется для гоночных и акробатических полетов.
* **RATTITUDE** — около центра правого стика работает как **STABILIZED**, а ближе к краям переходит в поведение **ACRO**.

## С использованием дополнительных датчиков

* **ALTCTL** (*Altitude*) — управление высотой с использованием барометра или другого датчика высоты.
* **POSCTL** (*Position*) — управление положением и скоростью движения. Использует барометр, GPS, компьютерное зрение или другие датчики. Это один из самых удобных режимов для обычных полетов.

## Автоматический полет

В автоматических режимах квадрокоптер выполняет программу полета и не управляется напрямую стиками пульта (происходит переключение в Position).

* **OFFBOARD** — управление полетом с внешнего компьютера, например с Raspberry Pi. Этот режим используется для автономных полетов из ROS 2.
* **AUTO.MISSION** — выполнение заранее загруженной миссии. Миссию можно загрузить через QGroundControl или по MAVLink.
* **AUTO.RTL** — автоматический возврат в точку взлета.
* **AUTO.LAND** — автоматическая посадка.

Дополнительная информация: [PX4 Flight Modes](https://docs.px4.io/main/en/flight_modes/).
