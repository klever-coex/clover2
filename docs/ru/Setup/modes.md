# Полётные режимы

**Режим** полётного контроллера PX4 определяет, как квадрокоптер реагирует на команды с пульта. Режимы обычно переключаются одним из тумблеров на аппаратуре управления.

Чтобы настроить полётные режимы:

1. В QGroundControl перейдите в панель *Vehicle Setup*.
2. В левом меню выберите *Flight Modes*.
3. Установите *Mode Channel* на переключатель **SwC** (*Channel 6*).
4. При необходимости установите *Emergency Kill Switch Channel* на переключатель **SwA** (*Channel 5*) — это аварийное выключение моторов (kill switch).
5. В пунктах Flight Mode 1-6 выберите нужные режимы для разных положений переключателя.

Рекомендуемые режимы:

* *Flight Mode 1*: **Stabilized**
* *Flight Mode 4*: **Altitude**
* *Flight Mode 6*: **Position**

После настройки переключайте тумблер на пульте управления и проверяйте в QGroundControl, что активируется правильный режим (он подсвечивается).

## Ручное управление

В ручных режимах пилот управляет квадрокоптером напрямую. GPS, компьютерное зрение и барометр не используются. Для таких режимов нужны хорошие навыки пилотирования.

* **STABILIZED** / **MANUAL** — стабилизация горизонтального положения. Пилот управляет газом, наклоном по тангажу и крену, а также угловой скоростью по рысканью. Дрон самостоятельно выравнивается по крену и тангажу, если отпустить стики.
* **ACRO** — управление газом и угловыми скоростями по тангажу, крену и рысканью. Чаще используется в гонках. Без хорошего опыта легко потерять контроль.
* **RATTITUDE** — около центра правого стика работает как **STABILIZED**, а ближе к краям переходит в поведение **ACRO**.

## Режимы с использованием дополнительных датчиков

* **ALTCTL** (*Altitude*) — управление скоростью изменения высоты полёта, углами по тангажу и крену и угловой скоростью по рысканью. Используется барометр (или иной датчик высоты) для удержания высоты.
* **POSCTL** (*Position*) — управление положением и скоростью движения. Использует барометр, GPS, компьютерное зрение или другие датчики. Это один из самых удобных режимов для обычных полётов.

## Автоматический полёт

В автоматических режимах дрон выполняет программу полёта и не управляется напрямую стиками пульта управления.

* **OFFBOARD** — управление полётом с внешнего компьютера, например с Raspberry Pi. Этот режим используется для автономных полётов, в том числе с применением ROS 2.
* **AUTO.MISSION** — выполнение заранее загруженной полётной миссии. Миссию можно загрузить через QGroundControl или по MAVLink.
* **AUTO.RTL** — автоматический возврат в точку взлёта: дрон поднимается (при необходимости), летит к точкеи выполняет посадку (поведение можно настроить параметрами).
* **AUTO.LAND** — автоматическая посадка в текущей точке. Полезно при потере сигнала или для безопасного завершения полёта.

Дополнительная информация: [PX4 Flight Modes](https://docs.px4.io/main/en/flight_modes/).
