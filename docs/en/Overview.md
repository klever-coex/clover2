Klever is a modern educational and research platform for a programmable quadcopter, designed for rapid prototyping, learning, and developing autonomous drones.

The platform combines reliable modern hardware with an up-to-date software stack. At its core is a high-performance flight controller based on STM32H7 with PX4, providing high-frequency computation and stable execution of control algorithms, along with a Raspberry Pi 5 as the onboard computer for running ROS 2 nodes, computer vision, and custom user logic.

The updated Klever platform focuses on a modern robotics ecosystem:

- ROS 2 — for building scalable, distributed, and reliable control systems
- Docker — for isolated environments, reproducible builds, and easy deployment
- Ready-made containers and project templates — so you can start working within minutes
- Open-source code — for learning and customization

This approach makes the platform equally convenient for beginners, who can quickly launch their first flight and start programming without complex setup thanks to the preconfigured system image, examples, and detailed documentation, and for advanced developers, who need a stable and reliable foundation for complex projects — including autonomous navigation, computer vision tasks, drone swarm development, as well as research and industrial applications.

The package includes a fully preconfigured Raspberry Pi image with all necessary software, libraries, and development tools. All software and documentation are open-source and available on [GitFlic](https://gitflic.ru/project/klever-coex/Klever2).
